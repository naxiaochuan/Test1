java中的8中基本数据类型：
	整型：    byte short int long
	浮点型：float double
	字符型：char
	布尔型：boolean(true/false)
	
标识符：类、方法、变量的名称
	必须以字母、下划线、美元符号开始
	对大小写敏感
	不能是java中的关键字
	
声明变量：类型   标识符(变量名);
变量初始化：标识符 = 值;
声明变量并赋值：类型  变量名 = 值;

++和--:
	当变量未参与其它运算时，++在前和++后无区别
	当变量有参与其它运算时，++在前标识优先级高，先自增后参与其它运算，++在后标识优先级低，先参与其它运算后自增
	--同理

自动类型转换：小转大
	类型兼容
	目标类型>原类型
	语法:目标类型 变量名 = 原类型数据;	
	
强转类型转换：大转小
	类型兼容
	目标类型<原类型
	语法:目标类型 变量名 = (目标类型)原类型数据;	
	***强转类型转换有可能造成数据精度的丢失
